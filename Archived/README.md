# Progetto AI COMUNE

Progetto di intelligenza artificiale a supporto dei processi amministrativi comunali.

## Descrizione

Il progetto esplora l'applicazione di modelli AI per l'analisi e la gestione di procedimenti amministrativi in ambito comunale, con focus su:

- **Segreteria Generale** — analisi e revisione normativa dei procedimenti
- **Servizi Sociali** — gestione e revisione dei procedimenti sociali

## Struttura del repository

| File | Contenuto |
|------|-----------|
| `Segreteria-Generale.md` | Procedimenti della Segreteria Generale |
| `Servizi-Sociali.md` | Procedimenti dei Servizi Sociali |
| `GOLDEN SAMPLE-*-Input.md` | Dati di input di riferimento (golden sample) |
| `GOLDEN SAMPLE-*-Output.md` | Output attesi di riferimento |
| `GOLDEN SAMPLE-*-RevisioneNormativa.md` | Revisioni normative di riferimento |
| `COMUNE-SEGRETERIA-GENERALE-*.md` | Configurazioni e versioni del sistema |
